ArtTap UI/UX Patterns

What is this website

favorite This is an ArtTap UI/UX design pattern library for review and comment. The pages should capture the UI interactions, behaviors, layout, and responsivity. This version of the repo uses MaterializeCSS framework (Why?).

Github repo for this site (you'll need an invitation from @wflaschka).

Art Tap Pages for Code (Shared Google Drive)

Site dependencies: Git, Node, Gulp, Sass (SCSS), Less, Browsersync.io, CSS, Flexbox HTML, Bulma.io, Spectre CSS, Semantic-UI, MaterializeCSS, Javascript, JQuery, Vue2, various Sass mixins

Optimization notes: When it's time, animate.scss can be greatly trimmed from its 560k size.

Why MaterializeCSS?

This version of the AT-UI-UX repo (v2) uses MaterializeCSS instead of Bulma because it has a wider range of components and pre-built interactions. The overhead for Bulma (e.g. getting dropdown menus to work, borrowing from Spectre.css, etc.) started to outweigh the benefits of having the lean framework.
